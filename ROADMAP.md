# Roadmap

This roadmap captures where Mathe-Schatzreise is heading. Nothing here is a
promise of a date - it is a shared wish list. Ideas, feedback and pull requests
are very welcome.

## Near term

- **Sound effects** - optional, gentle audio cues for correct answers, streaks
  and treasure chests (off by default, fully local).
- **Accessibility polish** - expanded screen-reader labels and a
  dyslexia-friendly font option.
- **More complete translations** - full coverage for every listed language,
  with native-speaker review.

## Mid term

- **Parent dashboard** - a local, private overview of practice time, accuracy
  trends and favorite modes (stored on-device only).
- **Difficulty analytics** - a smarter adaptive engine that spots which fact
  families need more practice.
- **Certificates** - printable, shareable "treasure explorer" certificates.

## Long term

- **More worlds** - themed maps (jungle, space, ocean) with new visuals.
- **Expanded multilingual mode** - spoken prompts and localized number names.
- **Classroom mode** - teacher-friendly group play, rotating challenges, and a
  shared (still local-first) progress board.

## Under consideration

- Custom number-range settings for differentiated learning.
- A "practice this fact" focused drill mode.
- Themed treasure collections to encourage return visits.

Have an idea? Open a feature request - see [CONTRIBUTING.md](CONTRIBUTING.md).
